# DI금융 v6.7 개선 로그 (2026-03-12)

## 변경사항 요약

### 1. CMF/MFI 값 스크리너 UI 표시
- AssetCard 확장 시 지표 상세 그리드에 CMF, MFI(14) 타일 추가
- CMF: 양수(매집) 녹색, 음수(분산) 빨간색 컬러코딩
- MFI: 과매도(<20) 보라색, 과매수(>80) 빨간색 컬러코딩

### 2. MACD 다이버전스 방향 표시
- SignalTag 컴포넌트에 `macdDivType` prop 전달
- "MACD 상승 다이버전스" / "MACD 하락 다이버전스" 구분 표시
- 기존 "MACD 다이버전스" 레이블을 방향성 있는 레이블로 교체

### 3. 스크리닝 조건 확장
- 수급 & 거래량 섹션에 CMF 매집/분산, MFI 과매도/과매수 4개 조건 추가
- 모멘텀 & 추세 섹션에 ADX 강세/약세 추세 2개 조건 추가
- TAG_COLORS에 신규 6개 조건 색상 매핑 추가

### 4. BB 스퀴즈 Keltner Channel 고도화
- 기존: 밴드폭 52주 최저 비교 방식

- 폴백: Keltner 미충족 시 기존 밴드폭 방식도 병합 적용
- CONDITION_META 설명 업데이트
